1. Set Exact Visibility (Kaun dekh sakta hai)

  • Close Friends Only: Sirf aapke selected close friends ko dikhai degi post.
  • Mutuals: Sirf wo log dekh sakte hain jo aapko follow karte hain aur aap bhi unhe follow karte ho.
  • Anonymous Followers: Aise followers jinke naam ya profile details aapko nahi dikhte, unko bhi visibility set kar sakte ho.

2. Auto-Lock After Time (Samay ke baad automatic lock hona)

Aap apni post ya story ko aise set kar sakte ho ki wo kuch samay baad khud hi lock ho jaye, jaise ek "timed vault".

Example: Aap post karte ho aur set karte ho ki 3 ghante baad ye sirf mujhe hi visible rahe, ya 24 ghante ke baad post automatically lock ho jaye.

3. Unlockable Posts (Request karke access lena)

Kuch posts ko aap Unlockable bana sakte ho. Jab koi user aisi post dekhna chahega, to usse ek custom message bhejna padega request karne ke liye.

Aap decide karoge ki request accept karni hai ya nahi. Ye feature khas taur pe kaam ka hai:

  • Artists apne premium art work ko limited audience ke saath share kar sakte hain.
